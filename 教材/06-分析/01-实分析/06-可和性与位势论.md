## 第100章：可和性方法与矩阵变换

可和性理论（Summability Theory）研究为发散级数赋予"广义和"的系统方法及其间的关系，在Fourier级数求和、发散积分正则化和Dirichlet级数解析延拓中有广泛应用。

### 矩阵变换与正则性

**定义100.1**（矩阵变换）：设$A = (a_{mn})_{m,n \in \mathbb{N}}$为无穷矩阵。对数列$x = (x_n)$，定义$A$的矩阵变换为
$$(Ax)_m = \sum_{n=1}^\infty a_{mn} x_n$$
若对每个收敛序列$x = (x_n)$，$(Ax)_m$对所有$m$收敛且$\lim_{m \to \infty} (Ax)_m = \lim_{n \to \infty} x_n$，则称$A$为**正则可和矩阵**（或Toeplitz矩阵）。

**定理100.1**（Silverman-Toeplitz定理 — 正则可和性的充要条件）：矩阵$A = (a_{mn})$为正则可和的当且仅当：
(i) $\lim_{m \to \infty} a_{mn} = 0$（对所有$n$）；
(ii) $\lim_{m \to \infty} \sum_{n=1}^\infty a_{mn} = 1$；
(iii) $\sup_m \sum_{n=1}^\infty |a_{mn}| < \infty$（一致有界性）。

**证明**：必要性——分别取$x = e_n$（第$n$个标准基向量）、$x = (1,1,1,\dots)$以及利用Banach-Steinhaus一致有界原理作用于部分和算子。充分性——将收敛序列$x = x_n$写为$x_n = L + \varepsilon_n$（$L = \lim x_n$，$\varepsilon_n \to 0$），由条件(i)-(iii)推出$(Ax)_m \to L$。$\blacksquare$

**定义100.2**（Cesàro $(C,\alpha)$求和）：对$\alpha > -1$，级数$\sum a_k$的$(C,\alpha)$和定义为Cesàro平均的极限。命部分和$s_n = \sum_{k=0}^n a_k$，定义$A_n^\alpha = \binom{n+\alpha}{n}$（广义二项式系数），则$(C,\alpha)$平均为
$$S_n^\alpha = \frac{1}{A_n^\alpha} \sum_{k=0}^n A_{n-k}^{\alpha-1} s_k$$
若$\lim_{n \to \infty} S_n^\alpha = S$存在，称级数为$(C,\alpha)$可和、和为$S$。$\alpha = 0$即通常收敛，$\alpha = 1$即$(C,1)$（算术平均）。若$(C,\alpha)$可和，则对$\beta > \alpha$也$(C,\beta)$可和且和相同。

**定理100.2**（Cesàro正则性）：对$\alpha > -1$，$(C,\alpha)$求和法是正则的——若级数收敛，其$(C,\alpha)$和等于通常和。

**证明**：$(C,\alpha)$求和法对应的矩阵元素$a_{mn} = A_{m-n}^{\alpha-1} / A_m^\alpha$（$n \leq m$）否则为零。验证Silverman-Toeplitz三条件：(i)$a_{mn} \to 0$因$A_m^\alpha \sim m^\alpha / \Gamma(\alpha+1)$；(ii)$\sum a_{mn} = 1$由二项式系数恒等式；(iii)所有项非负故一致有界成立。$\blacksquare$

### Abel可和性

**定义100.3**（Abel可和性）：级数$\sum a_k$的**Abel和**定义为
$$A\text{-}\sum_{k=0}^\infty a_k = \lim_{r \to 1^-} \sum_{k=0}^\infty a_k r^k$$
若右边对$r \in [0,1)$收敛且$r \to 1^-$时极限存在。

**定理100.3**（Abel定理）：若级数$\sum a_k$收敛，则其Abel可和且和等于通常和。

**证明**：设$s_n = \sum_{k=0}^n a_k \to s$。利用分部求和（Abel变换）：
$$\sum_{k=0}^\infty a_k r^k = (1-r) \sum_{k=0}^\infty s_k r^k$$
对于固定的$r \in [0,1)$，$s_k \to s$配合$(1-r)\sum r^k = 1$，由Toeplitz定理即得极限为$s$。$\blacksquare$

**定理100.4**（Cesàro-Abel包含关系）：若级数是$(C,1)$可和的，则也是Abel可和的且和相等。反之不真——Abel可和严格包含$(C,1)$可和类。

**证明**：将$(C,1)$平均表为$\sigma_n = \sum_{k=0}^n (1 - \frac{k}{n+1}) a_k$，引入生成函数$f(r) = \sum a_k r^k$，则$f(r) = (1-r)^2 \sum_{n=0}^\infty (n+1) \sigma_n r^n$。由$\sigma_n$收敛配合Stolz定理知$f(r) \to \lim \sigma_n$。反例：$\sum (-1)^k k$的Abel和为$1/4$但$(C,1)$不可和。$\blacksquare$

### Tauber定理

**定义100.4**（Tauber条件）：级数$\sum a_k$满足**Tauber条件**若$a_n = o(1/n)$（即$n a_n \to 0$）；满足**Littlewood条件**若$a_n = O(1/n)$（$n a_n$有界）。

**定理100.5**（Tauber定理，1897）：若级数Abel可和且满足Tauber条件$a_n = o(1/n)$，则级数通常收敛。

**证明**：设Abel和为$s$。对任意$\varepsilon > 0$，取$N$使$r = 1 - 1/N$时$|f(r) - s| < \varepsilon$。利用$|n a_n| \leq \varepsilon$（$n$充分大），通过分部求和估计$|s_n - f(r)|$，可证$|s_n - s|$最终被$C\varepsilon$控制。$\blacksquare$

**定理100.6**（Littlewood Tauber定理，1911）：若级数Abel可和且满足$\sum_{k \leq n} k|a_k| = O(n)$（等价于$a_n = O(1/n)$），则级数通常收敛。

**证明**：Littlewood的原始证明使用复分析与Hardy-Littlewood极大函数；Kararnata (1930) 给出只用实分析方法的简化。核心是估计$s_n - \sigma_N$的差并利用多项式逼近。$\blacksquare$

### 位势论中的Riesz位势与Bessel位势

**定义100.5**（Bessel位势）：对$s \in \mathbb{R}$，定义**Bessel位势**$J_s = (I - \Delta)^{-s/2}$，其Fourier乘子为
$$\widehat{J_s f}(\xi) = (1 + |\xi|^2)^{-s/2} \hat{f}(\xi)$$
物理空间中$J_s f = G_s * f$，其中Bessel核$G_s(x)$在无穷远处指数衰减，由$G_s(x) = (4\pi)^{-s/2} \Gamma(s/2)^{-1} \int_0^\infty e^{-t - |x|^2/(4t)} t^{(s-n)/2} \frac{dt}{t}$给出。

**定理100.7**（Riesz位势与Bessel位势的关系）：Bessel位势$J_s$与Riesz位势$I_s$在原点附近行为相同，但在无穷远处$G_s$呈指数衰减而Riesz核仅多项式衰减：
$$G_s(x) \sim |x|^{s-n} \quad (|x| \to 0), \qquad G_s(x) \sim |x|^{(s-n-1)/2} e^{-|x|} \quad (|x| \to \infty)$$
因此$J_s : L^p \to L^p$对所有$1 \leq p \leq \infty$有界（Young不等式），而$I_\alpha$的$L^p$有界性需满足Hardy-Littlewood-Sobolev条件。

**证明**：比较两者的Fourier乘子$(1+|\xi|^2)^{-s/2}$与$|\xi|^{-s}$，在$|\xi| \to 0$时等价，在$|\xi| \to \infty$时前者衰减更快。物理空间行为由Tauber型渐近展开得出。$\blacksquare$

### Sobolev空间的位势刻画

**定义100.6**（Sobolev空间$H^s$的位势定义）：对$s \in \mathbb{R}$，
$$H^s(\mathbb{R}^n) = \{ f \in \mathcal{S}'(\mathbb{R}^n) : (I - \Delta)^{s/2} f \in L^2(\mathbb{R}^n) \}$$
即$H^s = (I - \Delta)^{-s/2} L^2 = J_s(L^2)$。范数定义为$\|f\|_{H^s} = \|(I - \Delta)^{s/2} f\|_{L^2} = \|J_{-s} f\|_{L^2}$。

**定理100.8**（Sobolev空间的位势等价）：对$s \geq 0$，$H^s(\mathbb{R}^n)$与经典Sobolev空间$W^{s,2}(\mathbb{R}^n)$等距同构。对$s < 0$，$H^s = (H^{-s})^*$（对偶空间）。

**证明**：对正整数$s = k$，由Plancherel定理知$\|f\|_{H^k}^2 = \int (1+|\xi|^2)^k |\hat{f}(\xi)|^2 d\xi$等价于$\sum_{|\alpha| \leq k} \|D^\alpha f\|_{L^2}^2$。对分数阶$s$，该定义为经典范数的插值。对偶性由Bessel位势算子的自伴性得出。$\blacksquare$

**定义100.7**（位势的极大函数估计）：对Riesz位势$I_\alpha f$，定义极大位势算子
$$M_\alpha f(x) = \sup_{r > 0} r^\alpha \frac{1}{|B(x,r)|} \int_{B(x,r)} |f(y)| dy$$

**定理100.9**（位势的极大函数控制）：对$0 < \alpha < n$，存在常数$C$使
$$I_\alpha(|f|)(x) \leq C M_\alpha f(x)$$
且$M_\alpha f$映射$L^p$到$L^q$（$1/q = 1/p - \alpha/n$，$1 < p < n/\alpha$），从而给出HLS不等式的另一证明。

**证明**：将Riesz核的卷积积分按半径二进分解，每层以球平均控制，叠加得极大函数上界。$M_\alpha$的$L^p \to L^q$有界性由Hardy-Littlewood极大函数的加权估计得出。$\blacksquare$

### 可和性方法在三角级数中的应用

**定理100.10**（Fejér定理 — $(C,1)$可和在Fourier级数中的应用）：设$f$为周期$2\pi$的Riemann可积函数。其Fourier级数的$(C,1)$（Cesàro）平均$\sigma_n(f)(x)$在$f$的每个连续点$x$处收敛于$f(x)$，在跳跃间断点处收敛于$(f(x^+) + f(x^-))/2$。

**证明**：$(C,1)$平均可表为$\sigma_n(f) = F_n * f$，其中Fejér核$F_n(t) = \frac{1}{n+1} \sum_{k=0}^n D_k(t) = \frac{1}{n+1} \frac{\sin^2((n+1)t/2)}{\sin^2(t/2)} \geq 0$。$F_n$为近似恒同（正性、积分为$1$、在$0$外逐点趋于$0$），故由Lebesgue控制收敛定理得逐点收敛。$\blacksquare$

**定理100.11**（Abel可和在Fourier级数中的应用 — Poisson求和）：$f$的Fourier级数的Abel平均即Poisson积分：
$$u(r, \theta) = \sum_{k=-\infty}^\infty \hat{f}(k) r^{|k|} e^{i k \theta} = \frac{1}{2\pi} \int_{-\pi}^\pi f(\theta - t) \frac{1 - r^2}{1 - 2r\cos t + r^2} dt$$
当$r \to 1^-$时，$u(r, \theta) \to f(\theta)$在Lebesgue点处成立。这提供了Dirichlet问题在单位圆盘上的解的Poisson积分表示。

**证明**：Abel平均的核为Poisson核$P_r(t) = (1-r^2)/(1-2r\cos t + r^2)$，满足近似恒同性质，其逐点收敛性完全类似Fejér定理的证明。$\blacksquare$

---
*可和性理论以Silverman-Toeplitz定理为正则性判据，Cesàro与Abel方法构成求和法的两大支柱。Littlewood Tauber定理揭示了Abel可和性在温和条件下蕴含通常收敛性。位势论方面，Riesz与Bessel位势的关系澄清了局部与全局正则性的差异，Sobolev空间的位势刻画$H^s = (I-\Delta)^{-s/2}L^2$则是现代PDE理论的基石。Fejér定理与Poisson求和在三角级数中的应用，体现了可和性方法从抽象的矩阵变换到具体的Fourier分析的自然桥梁。*
