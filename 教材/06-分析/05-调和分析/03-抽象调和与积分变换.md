## 第141章：局部紧Abel群上的调和分析

抽象调和分析将经典Fourier分析从$\mathbb{R}^n$推广到任意局部紧Abel群（LCA群），其核心是Pontryagin对偶理论。

### Haar测度与Fourier变换

**定理141.1**（Haar测度的存在唯一性，陈述）：设$G$为局部紧群，则存在$G$上非零左不变（右不变）Radon测度$\mu$，在对$G$为Abel群时左右不变：$\mu(gE) = \mu(E)$对所有$g \in G$与Borel集$E$。该测度在相差正常数倍的意义下唯一。称此测度为**Haar测度**。紧群时$\mu(G) < \infty$（通常归一化为概率测度）；离散群时$\mu$为计数测度。证明需用到Urysohn引理与Riesz表示定理。

**定义141.1**（对偶群与特征标）：设$G$为LCA群。$G$的**对偶群**$\widehat{G}$是所有连续群同态$\chi : G \to \mathbb{T}$（$\mathbb{T} = \{z \in \mathbb{C} : |z| = 1\}$）的集合，配以紧致开拓扑与逐点乘法构成LCA群。$\chi \in \widehat{G}$称为$G$的**特征标**。

基本例子：$\widehat{\mathbb{R}} \cong \mathbb{R}$（$\chi_\xi(x) = e^{2\pi i x\xi}$）；$\widehat{\mathbb{T}} \cong \mathbb{Z}$（$\chi_n(e^{i\theta}) = e^{in\theta}$）；$\widehat{\mathbb{Z}} \cong \mathbb{T}$；$\widehat{\mathbb{Z}/n\mathbb{Z}} \cong \mathbb{Z}/n\mathbb{Z}$。

**定义141.2**（LCA群上的Fourier变换）：对$f \in L^1(G)$（关于Haar测度$dx$），其**Fourier变换**$\hat{f} : \widehat{G} \to \mathbb{C}$定义为
$$\hat{f}(\chi) = \int_G f(x) \overline{\chi(x)} dx$$
$\hat{f}$是$\widehat{G}$上的连续函数（Riemann-Lebesgue引理：$\hat{f} \in C_0(\widehat{G})$），且$\|\hat{f}\|_\infty \leq \|f\|_1$。

**定理141.2**（Pontryagin对偶定理）：存在拓扑群的自然同构$G \cong \widehat{\widehat{G}}$，由赋值映射$x \mapsto (\chi \mapsto \chi(x))$给出。

**证明**：紧群情形用Peter-Weyl理论与矩阵元的完备性；离散群情形由$\widehat{\widehat{\mathbb{Z}}} \cong \widehat{\mathbb{T}} \cong \mathbb{Z}$验证。一般LCA群通过紧群与$\mathbb{R}^n$的投影极限分解归结为这两种情形。$\blacksquare$

**定理141.3**（LCA群上的Plancherel定理）：设$G$为LCA群，Haar测度$dx$。存在$\widehat{G}$上的唯一Haar测度$d\chi$（Plancherel测度），使得Fourier变换可延拓为$L^2(G)$到$L^2(\widehat{G})$的酉算子：
$$\|f\|_{L^2(G)} = \|\hat{f}\|_{L^2(\widehat{G})}, \quad f \in L^1(G) \cap L^2(G)$$
逆变换由$f(x) = \int_{\widehat{G}} \hat{f}(\chi) \chi(x) d\chi$给出（$f \in L^1(\widehat{G})$时逐点成立）。

**证明**：利用幺模群上卷积代数$L^1(G)$的Gelfand变换理论，将Fourier变换视为乘法算子在谱上的表示，Plancherel测度对应Plancherel公式中的谱测度。$\blacksquare$

**定理141.4**（Bochner定理 — LCA群版）：$G$上的连续函数$\varphi$为正定型（即$\sum_{i,j} c_i \overline{c_j} \varphi(x_i^{-1}x_j) \geq 0$）当且仅当存在$\widehat{G}$上唯一正有限Borel测度$\mu$使
$$\varphi(x) = \int_{\widehat{G}} \chi(x) d\mu(\chi)$$
$\mu$称为$\varphi$的谱测度，是自协方差函数Wiener-Khinchin关系的抽象化。

**证明**：正定性推出$\varphi(0) < \infty$且$\varphi$为$L^1(G)$卷积代数上的正线性泛函，由Gelfand-Naimark-Segal构造与Riesz表示定理得测度$\mu$。唯一性由Fourier-Stieltjes变换的单射性保证。$\blacksquare$

**定理141.5**（Poisson求和公式的抽象版本）：设$G$为LCA群，$H \subset G$为闭子群。记$H^\perp = \{\chi \in \widehat{G} : \chi|_H \equiv 1\}$为$H$的零化子（$\widehat{G}$的闭子群，同构于$\widehat{G/H}$）。Haar测度适当归一化后，对Schwartz-Bruhat函数$f$有
$$\int_H f(h) dh = \int_{H^\perp} \hat{f}(\chi) d\chi$$
经典Poisson公式为特例：$G = \mathbb{R}$，$H = \mathbb{Z}$，$H^\perp = \mathbb{Z}$，即$\sum_{n \in \mathbb{Z}} f(n) = \sum_{n \in \mathbb{Z}} \hat{f}(n)$。

**证明**：定义周期化算子$P f(x) = \sum_{h \in H} f(xh)$，则$P f \in L^1(G/H)$。在商群$G/H$上展开Fourier级数，利用$\widehat{G/H} \cong H^\perp$即得求和公式。$\blacksquare$

## 第142章：积分变换深化

积分变换以积分核为桥梁，在线性PDE求解、信号处理和成像科学中是不可或缺的工具。

### Mellin变换

**定义142.1**（Mellin变换）：对$f : (0, \infty) \to \mathbb{C}$，
$$\mathcal{M}[f](s) = \int_0^\infty f(x) x^{s-1} dx$$
定义域为复平面上使积分收敛的$s$的带形区域$\{\sigma_1 < \operatorname{Re}(s) < \sigma_2\}$。逆变换为
$$f(x) = \frac{1}{2\pi i} \int_{c - i\infty}^{c + i\infty} \mathcal{M}[f](s) x^{-s} ds$$
其中$c$在收敛带形内。

**定理142.1**（Mellin变换与Fourier变换的关系）：通过指数换元$x = e^t$，Mellin变换等价于Fourier变换：
$$\mathcal{M}[f](s) = \int_{-\infty}^\infty f(e^t) e^{i(-is)t} e^t dt = \mathcal{F}[g](\xi)$$
其中$g(t) = f(e^t) e^t$，$\xi = -is$（即$s = c + i\xi$时）。因此所有Mellin变换性质可直接由Fourier变换性质翻译得到。

**证明**：代换$x = e^t$，$dx = e^t dt$，$x^{s-1} = e^{(s-1)t}$，故
$$\mathcal{M}[f](s) = \int_{-\infty}^\infty f(e^t) e^{st} dt = \int_{-\infty}^\infty [f(e^t) e^t] e^{i(-is)t} dt$$
在$s = c + i\omega$时化为Fourier变换。$\blacksquare$

**定理142.2**（Mellin卷积与尺度不变性）：
(i) $\mathcal{M}[f(ax)](s) = a^{-s} \mathcal{M}[f](s)$（尺度不变性）；
(ii) Mellin卷积$(f \star g)(x) = \int_0^\infty f(x/y) g(y) \frac{dy}{y}$满足$\mathcal{M}[f \star g] = \mathcal{M}[f] \cdot \mathcal{M}[g]$。

**证明**：(i)由变量代换$x \mapsto x/a$直接得出。(ii)类似Fourier变换的卷积定理——写Mellin卷积为Fourier卷积通过指数换元。$\blacksquare$

### Laplace变换与Bromwich反演

**定义142.2**（Laplace变换）：对$f : [0, \infty) \to \mathbb{C}$，
$$\mathcal{L}[f](s) = \int_0^\infty f(t) e^{-st} dt$$
定义域为$\{s \in \mathbb{C} : \operatorname{Re}(s) > \sigma_c\}$，其中$\sigma_c = \inf\{\sigma : \int_0^\infty |f(t)| e^{-\sigma t} dt < \infty\}$为**收敛横坐标**。若$\sigma_c = -\infty$称$f$为指数型函数全体。

**定理142.3**（Laplace反演公式 — Bromwich积分）：若$f$分段光滑且在$[0,\infty)$上指数增长，则对$c > \sigma_c$（且$c$大于$f$的Laplace变换所有奇点的实部），
$$f(t) = \frac{1}{2\pi i} \int_{c - i\infty}^{c + i\infty} \mathcal{L}[f](s) e^{st} ds$$
即由$\mathcal{L}[f]$在复平面右半平面的取值沿铅垂线($\operatorname{Re}(s) = c$)的围道积分恢复$f(t)$。

**证明**：将Laplace变换写为Fourier变换：设$g(t) = f(t) e^{-ct}$，则$\mathcal{L}[f](c + i\omega) = \mathcal{F}[g](\omega/2\pi)$。由Fourier反演公式
$$g(t) = \frac{1}{2\pi} \int_{-\infty}^\infty \mathcal{F}[g](\omega) e^{i\omega t} d\omega = \frac{1}{2\pi i} \int_{c-i\infty}^{c+i\infty} \mathcal{L}[f](s) e^{st} ds$$
两边同乘$e^{ct}$即得。$\blacksquare$

### Hankel变换

**定义142.3**（Hankel变换）：对$\mathbb{R}^n$上径向函数$f(x) = f_0(|x|)$，其Fourier变换也是径向的。Hankel变换是径向函数的Fourier变换在极坐标下的归约。记$\nu = n/2 - 1$，
$$\mathcal{H}_\nu [f_0](\rho) = (2\pi)^{n/2} \rho^{-\nu} \int_0^\infty f_0(r) J_\nu(2\pi \rho r) r^{n/2} dr$$
其中$J_\nu$为Bessel函数。对逆变换，$\mathcal{H}_\nu^{-1} = \mathcal{H}_\nu$（对适当函数类）。

**定理142.4**（Hankel变换与Fourier变换的关系）：对$f(x) = f_0(|x|) \in L^1(\mathbb{R}^n)$，
$$\hat{f}(\xi) = \mathcal{H}_\nu [f_0](|\xi|)$$
即径向函数的Fourier变换化为Hankel变换。对$L^2(\mathbb{R}_+, r^{n-1}dr)$上函数，$\mathcal{H}_\nu$为酉算子。

**证明**：将Fourier积分$\hat{f}(\xi) = \int_{\mathbb{R}^n} f_0(|x|) e^{-2\pi i x \cdot \xi} dx$转为极坐标，角度部分积分为Bessel函数$J_\nu$，得Hankel积分表示。$\blacksquare$

### 小波变换与多尺度分析

**定义142.4**（连续小波变换）：设小波母函数$\psi \in L^2(\mathbb{R})$满足容许条件$C_\psi = \int_{-\infty}^\infty |\hat{\psi}(\xi)|^2 |\xi|^{-1} d\xi < \infty$。对$f \in L^2(\mathbb{R})$，定义连续小波变换
$$\mathcal{W}_\psi f(a, b) = \frac{1}{\sqrt{|a|}} \int_{-\infty}^\infty f(t) \overline{\psi\left(\frac{t-b}{a}\right)} dt$$
其中$a \in \mathbb{R} \setminus \{0\}$为尺度参数，$b \in \mathbb{R}$为平移参数。反演公式为
$$f(t) = \frac{1}{C_\psi} \int_{-\infty}^\infty \int_{-\infty}^\infty \mathcal{W}_\psi f(a, b) \frac{1}{\sqrt{|a|}} \psi\left(\frac{t-b}{a}\right) \frac{da\,db}{a^2}$$

**定理142.5**（连续小波变换的等距性）：$\|\mathcal{W}_\psi f\|_{L^2(\mathbb{R}_+^* \times \mathbb{R}, a^{-2} da db)} = \sqrt{C_\psi} \|f\|_{L^2(\mathbb{R})}$——即小波变换是$L^2$到加权$L^2$的等距嵌入（模常数因子）。

**证明**：对固定尺度$a$，Fourier变换将伸缩平移化为乘子：$\mathcal{F}[\psi_{a,b}](\xi) = \sqrt{|a|} e^{-2\pi i b \xi} \hat{\psi}(a\xi)$。于是小波系数可写作Fourier侧的内积。由Plancherel定理与容许条件的定义得等距性。$\blacksquare$

**定义142.5**（多尺度分析与正交小波基）：$L^2(\mathbb{R})$的**多尺度分析**（MRA）是一列闭子空间$\{V_j\}_{j \in \mathbb{Z}}$满足：
(i) $\cdots \subset V_{-1} \subset V_0 \subset V_1 \subset \cdots$（嵌套性）；
(ii) $\overline{\bigcup_{j} V_j} = L^2(\mathbb{R})$（稠密性）且$\bigcap_j V_j = \{0\}$（分离性）；
(iii) $f \in V_j \iff f(2 \cdot) \in V_{j+1}$（二进伸缩性）；
(iv) 存在尺度函数$\varphi \in V_0$使$\{\varphi(\cdot - k) : k \in \mathbb{Z}\}$为$V_0$的规范正交基。

由MRA可构造小波函数$\psi$使$\{\psi_{j,k}(x) = 2^{j/2} \psi(2^j x - k) : j, k \in \mathbb{Z}\}$为$L^2(\mathbb{R})$的规范正交基——这就是**离散小波变换**的正交基表达。Daubechies（1988）证明了存在具紧支集的任意阶光滑小波。

**定理142.6**（MRA的尺度方程与两尺度关系）：尺度函数$\varphi$满足**尺度方程**$\varphi(x) = \sum_{k} h_k \varphi(2x - k)$，其中$\{h_k\}$为低通滤波器系数。小波函数$\psi(x) = \sum_k g_k \varphi(2x - k)$由高通滤波器$\{g_k\}$生成，且$g_k = (-1)^k \overline{h_{1-k}}$（正交小波时）。

**证明**：由$V_0 \subset V_1$，$\varphi \in V_0$可展为$\{\varphi(2 \cdot - k)\}$的线性组合即得尺度方程。由$\psi \in W_0 = V_1 \ominus V_0$可得高通滤波表示。正交性条件给出滤波器系数的精确关系。$\blacksquare$

---
*抽象调和分析以Pontryagin对偶为核心，将Fourier变换提升至LCA群层面，Plancherel定理与Poisson求和公式在此框架下获得统一表达。积分变换方面，Mellin变换经指数换元等同Fourier变换，Laplace变换以Bromwich积分反演，Hankel变换处理径向函数的Fourier分析。小波变换以多尺度分析为理论骨架，将时频局部化推向极致，弥补了Fourier分析的全局性局限。*
