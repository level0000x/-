# 从公理到整个数学体系

> 按严格依赖链组织的数学全书。从 ZFC 公理出发，逐层构建整个数学体系。
> 共 13 个领域，379 章，约 110 个文件。

## 全局依赖图

```
01-基础 → 02-几何
        → 03-函数与微积分 → 04-离散数学
                          → 05-代数 → 06-分析
                                    → 07-拓扑与几何
                                    → 08-代数几何与数论
                                    → 13-数学物理
                          → 09-微分方程与动力系统
                          → 10-概率与统计 → 12-最优化与控制
                          → 11-数值与计算
```

## 使用说明

- 每个 `.md` 文件含 1-6 章，按依赖顺序排列
- 每个目录的 `_index.md` 说明前置依赖与内容概要
- 全书按目录/文件名排序即为正确阅读顺序
- 编译时合并 `教材/` 下所有 `.md`（排除 `_index.md`）
