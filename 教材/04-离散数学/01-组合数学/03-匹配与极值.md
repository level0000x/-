## 第38章：极值集合论（Extremal Set Theory）
极值集合论研究集合系在一定限制条件下能达到的最大尺寸或最优结构。**核心结果**：（1）**Sperner 定理**（1928年）——$n$ 元集合的子集族中，若不存在两个集合有包含关系（即族是反链），则该族的最大尺寸为 $\binom{n}{\lfloor n/2 \rfloor}$（取自最中间层的所有子集）。这是极值集合论的奠基性结果，由对称链分解证明；（2）**Erdős-Ko-Rado 定理**（1961年）——设 $\mathcal{F}$ 是 $\{1,\ldots,n\}$ 上所有大小为 $k$ 的子集组成的族，若 $n \geq 2k$ 且 $\mathcal{F}$ 中任意两个集合的交非空，则 $|\mathcal{F}| \leq \binom{n-1}{k-1}$（即包含某个固定元素的所有 $k$-子集）；（3）**Kruskal-Katona 定理**——在保持 $k$-一致超图边数的约束下，由初始段（colexicographic order）给出影子（$(k-1)$-子集）的最小基数；（4）**有限域上的极值集合论**——Frankl-Wilson 定理用代数方法（多项式空间的维数限制）证明集合族的 Ramsey 型上界，在 Frankl-Wilson 对 Borsuk 猜想的否定解决（1993年）中有决定性应用。**新兴方向**：$L$-交族在编码理论和 Ramsey 理论中的交叉应用；加性组合极值理论（Roth 定理的集合推广）；无标度网络中最大度分布的极值特征。

## Ch 39 概率
### 29.1 随机事件与概率

#### 29.1.1 随机试验与样本空间

> **直觉与动机** 概率论研究的是不确定性。在自然界和日常生活中，许多现象的结果在事先无法确定——掷骰子的点数、明年的降雨量、放射性原子的衰变时刻。概率论为这种不确定性提供了严格的数学框架。

**定义 29.1（随机试验）** 满足以下条件的试验称为**随机试验**：
1. 可以在相同条件下重复进行；
2. 所有可能结果事先已知；
3. 每次试验的具体结果事先无法确定。

**定义 29.2（样本空间）** 随机试验所有可能结果的集合称为**样本空间**，记作 $\Omega$。样本空间中的元素称为**样本点**，记作 $\omega$。

**定义 29.3（事件）** 样本空间 $\Omega$ 的子集称为**事件**。特别地：
- **必然事件**：$\Omega$（一定发生的事件）
- **不可能事件**：$\varnothing$（一定不发生的事件）
- **基本事件**：只包含一个样本点的事件

#### 29.1.2 事件的关系与运算

设 $A, B$ 为事件：
- **包含**：$A \subset B$ 表示"$A$ 发生则 $B$ 必发生"
- **和事件**：$A \cup B$ 表示"$A$ 与 $B$ 至少有一个发生"
- **积事件**：$A \cap B$ 表示"$A$ 与 $B$ 同时发生"
- **互斥**：$A \cap B = \varnothing$ 表示"$A$ 与 $B$ 不能同时发生"
- **对立事件**：$\bar{A} = \Omega \setminus A$ 表示"$A$ 不发生"

#### 29.1.3 概率的公理化定义

**定义 29.3'（$\sigma$-代数）** 设 $\Omega$ 为非空集合，$\mathcal{F}$ 是 $\Omega$ 的子集族。如果 $\mathcal{F}$ 满足以下三个条件，则称 $\mathcal{F}$ 为 $\Omega$ 上的一个 **$\sigma$-代数**（或 **$\sigma$-域**）：
1. $\Omega \in \mathcal{F}$（全集属于 $\mathcal{F}$）
2. 若 $A \in \mathcal{F}$，则 $\bar{A} = \Omega \setminus A \in \mathcal{F}$（对补运算封闭）
3. 若 $A_1, A_2, \ldots \in \mathcal{F}$，则 $\bigcup_{i=1}^{\infty} A_i \in \mathcal{F}$（对可数并运算封闭）

> **$\sigma$-代数的意义** $\sigma$-代数是概率论中"可测事件"的精确定义。并非 $\Omega$ 的所有子集都能被赋予概率（在某些情形下，如选择公理构造的不可测集），$\sigma$-代数限定了我们能够讨论概率的事件范围。条件 1 确保必然事件可测，条件 2 确保事件的否定可测，条件 3 确保可数个事件的并可测——这恰好对应概率公理系统的需要。

**定义 29.4（概率）** 设 $\Omega$ 为样本空间，$\mathcal{F}$ 为 $\Omega$ 上的一个 $\sigma$-代数（见定义 29.3'），函数 $P: \mathcal{F} \to [0, 1]$ 称为**概率**，如果满足以下三条公理（柯尔莫哥洛夫公理）：

1. **非负性**：对任意事件 $A$，$P(A) \geq 0$
2. **规范性**：$P(\Omega) = 1$
3. **可列可加性**：若 $A_1, A_2, \ldots$ 是两两互斥的事件，则
$$P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i)$$

> **有限可加性** 从可列可加性可以推出**有限可加性**：若 $A_1, A_2, \ldots, A_n$ 两两互斥，则 $P(A_1 \cup A_2 \cup \cdots \cup A_n) = P(A_1) + P(A_2) + \cdots + P(A_n)$。这只需令 $A_{n+1} = A_{n+2} = \cdots = \varnothing$ 即可由可列可加性得到。有限可加性在实际计算中更常用，但可列可加性在处理无穷样本空间和极限运算时不可或缺。



> **直觉与动机** 这三条公理是柯尔莫哥洛夫在1933年提出的，它们将概率论建立在严格的测度论基础之上。非负性保证概率不为负，规范性保证"某事发生"的概率为1，可列可加性保证互斥事件的并的概率等于各自概率之和——这正是频率解释中"比例相加"的数学化。

**定理 29.1（概率的基本性质）**
1. $P(\varnothing) = 0$
2. $P(\bar{A}) = 1 - P(A)$
3. $P(A \cup B) = P(A) + P(B) - P(A \cap B)$（容斥原理）
4. 若 $A \subset B$，则 $P(A) \leq P(B)$

**证明**：
1. 取 $A_1 = \Omega, A_2 = A_3 = \cdots = \varnothing$，由公理3（可列可加性）得 $P(\Omega) = P(\Omega) + \sum_{i=2}^{\infty} P(\varnothing)$。因 $P$ 非负，该级数收敛仅当 $P(\varnothing) = 0$。

2. $A \cup \bar{A} = \Omega$，$A \cap \bar{A} = \varnothing$。由公理3：$P(\Omega) = P(A) + P(\bar{A})$，故 $P(\bar{A}) = 1 - P(A)$。

3. $A \cup B = A \cup (B \setminus A)$（不交并），$B = (A \cap B) \cup (B \setminus A)$（不交并）。故：
$$P(A \cup B) = P(A) + P(B \setminus A)$$
$$P(B) = P(A \cap B) + P(B \setminus A)$$
消去 $P(B \setminus A)$：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$。

4. $B = A \cup (B \setminus A)$（不交并），$P(B) = P(A) + P(B \setminus A) \geq P(A)$。
$\blacksquare$

### 29.2 古典概型与几何概型

#### 29.2.1 古典概型

> **直觉与动机** 古典概型是最简单的概率模型，适用于"等可能"的情形——如掷骰子、抽扑克牌。其核心假设是每个基本事件发生的可能性相同。

**定义 29.5（古典概型）** 如果随机试验满足：
1. 样本空间 $\Omega$ 只有有限个样本点；
2. 每个基本事件等可能发生（即 $P(\{\omega_i\}) = \frac{1}{|\Omega|}$）；

则称为**古典概型**。事件 $A$ 的概率为：
$$P(A) = \frac{|A|}{|\Omega|} = \frac{\text{事件 } A \text{ 包含的样本点数}}{\text{样本空间的样本点总数}}$$

**例** 从 $1, 2, \ldots, 10$ 中随机取一个数，求取到偶数的概率。

**解** $\Omega = \{1, 2, \ldots, 10\}$，$|\Omega| = 10$。$A = \{2, 4, 6, 8, 10\}$，$|A| = 5$。$P(A) = \frac{5}{10} = \frac{1}{2}$。

#### 29.2.2 几何概型

**定义 29.6（几何概型）** 如果随机试验的结果可以用某个几何区域 $\Omega$ 中的点表示，且等可能性意味着"落在 $\Omega$ 内任何子区域的概率与该子区域的测度（长度、面积或体积）成正比"，则事件 $A$ 的概率为：
$$P(A) = \frac{\mu(A)}{\mu(\Omega)}$$

其中 $\mu$ 表示相应的测度（长度、面积或体积）。

**例（贝特朗悖论的启示）** 在单位圆内随机取一条弦，求弦长超过 $\sqrt{3}$ 的概率。此问题的不同答案取决于"随机"的具体含义（如何参数化弦的选取），这说明概率模型的精确定义至关重要。

### 29.3 条件概率与独立性

#### 29.3.1 条件概率

> **直觉与动机** 条件概率回答的问题是："在已知事件 $B$ 发生的条件下，事件 $A$ 发生的概率是多少？"例如，已知今天是阴天，明天下雨的概率是多少？条件概率通过缩小样本空间来更新我们对不确定性的判断。

**定义 29.7（条件概率）** 设 $A, B$ 为事件，$P(B) > 0$。在事件 $B$ 已发生的条件下，事件 $A$ 发生的**条件概率**定义为：
$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

**定理 29.2（乘法公式）** 若 $P(B) > 0$，则：
$$P(A \cap B) = P(B) \cdot P(A|B)$$

一般地，若 $P(A_1 A_2 \cdots A_{n-1}) > 0$（以下简写 $A_1 A_2 \cdots A_k = A_1 \cap A_2 \cap \cdots \cap A_k$），则：
$$P(A_1 A_2 \cdots A_n) = P(A_1) \cdot P(A_2|A_1) \cdot P(A_3|A_1 A_2) \cdots P(A_n|A_1 A_2 \cdots A_{n-1})$$

**证明**：由条件概率定义，$P(A \cap B) = P(B) \cdot P(A|B)$。一般情形对 $n$ 归纳可得。$\blacksquare$

#### 29.3.2 全概率公式与贝叶斯公式

**定理 29.3（全概率公式）** 设 $B_1, B_2, \ldots, B_n$ 是样本空间 $\Omega$ 的一个**划分**（即 $B_i$ 两两互斥且 $\bigcup_{i=1}^{n} B_i = \Omega$），且 $P(B_i) > 0$，则对任意事件 $A$：
$$P(A) = \sum_{i=1}^{n} P(B_i) P(A|B_i)$$

**证明**：$A = A \cap \Omega = A \cap \left(\bigcup_{i=1}^{n} B_i\right) = \bigcup_{i=1}^{n} (A \cap B_i)$（不交并）。

由概率的可加性：$P(A) = \sum_{i=1}^{n} P(A \cap B_i) = \sum_{i=1}^{n} P(B_i) P(A|B_i)$。$\blacksquare$

> **全概率公式的意义** 全概率公式体现了"由因溯果"的思想：将事件 $A$ 的概率分解为各种可能原因 $B_i$ 下 $A$ 发生的条件概率的加权平均。这在医学诊断、质量检测等领域有广泛应用。

**定理 29.4（贝叶斯公式）** 在定理 29.3 的条件下，若 $P(A) > 0$，则：
$$P(B_j|A) = \frac{P(B_j) P(A|B_j)}{\sum_{i=1}^{n} P(B_i) P(A|B_i)}$$

**证明**：由条件概率定义和全概率公式：
$$P(B_j|A) = \frac{P(A \cap B_j)}{P(A)} = \frac{P(B_j) P(A|B_j)}{\sum_{i=1}^{n} P(B_i) P(A|B_i)}$$
$\blacksquare$

> **贝叶斯公式的意义** 贝叶斯公式实现了"由果溯因"的推理：已知结果 $A$ 发生了，反推各原因 $B_j$ 成立的概率。$P(B_j)$ 称为**先验概率**（实验前的判断），$P(B_j|A)$ 称为**后验概率**（获得新信息后的更新判断）。贝叶斯公式是现代统计学、机器学习和人工智能的核心理论基础之一。

#### 29.3.3 事件的独立性

**定义 29.8（独立事件）** 若 $P(A \cap B) = P(A) \cdot P(B)$，则称事件 $A$ 与 $B$ **相互独立**。

> **直觉与动机** 独立性的直观含义是"一个事件的发生不影响另一个事件发生的概率"。例如，掷两枚硬币，第一枚的结果不影响第二枚。注意独立性不等于互斥——互斥的两个事件不可能同时发生，而独立的两个事件可以同时发生。

**定理 29.5** 若 $A$ 与 $B$ 独立，则：
1. $A$ 与 $\bar{B}$ 独立
2. $\bar{A}$ 与 $B$ 独立
3. $\bar{A}$ 与 $\bar{B}$ 独立

**证明**：1. $P(A \cap \bar{B}) = P(A) - P(A \cap B) = P(A) - P(A)P(B) = P(A)[1 - P(B)] = P(A)P(\bar{B})$。

2 和 3 类似证明。$\blacksquare$

### 29.4 随机变量与常见分布

#### 29.4.1 随机变量

> **直觉与动机** 随机变量是将随机试验的结果"数值化"的工具。例如，掷骰子的结果可以表示为随机变量 $X$（取值 $1, 2, \ldots, 6$）。引入随机变量后，我们可以用函数、积分等分析工具来研究概率问题。

**定义 29.9（随机变量）** 定义在样本空间 $\Omega$ 上的实值函数 $X: \Omega \to \mathbb{R}$ 称为**随机变量**。

**定义 29.10（离散型随机变量的分布列）** 设离散型随机变量 $X$ 的可能取值为 $x_1, x_2, \ldots$，则 $P(X = x_k) = p_k$（$k = 1, 2, \ldots$）称为 $X$ 的**分布列**（或概率分布）。其中 $p_k \geq 0$ 且 $\sum_k p_k = 1$。

#### 29.4.2 数学期望

**定义 29.11（数学期望）** 设离散型随机变量 $X$ 的分布列为 $P(X = x_k) = p_k$，则 $X$ 的**数学期望**（或均值）定义为：
$$E(X) = \sum_{k} x_k p_k$$

（要求级数绝对收敛。）

> **直觉与动机** 数学期望是随机变量的"加权平均值"，它代表了随机变量在大量重复试验中的平均表现。例如，掷一枚均匀骰子的期望值为 $\frac{1+2+3+4+5+6}{6} = 3.5$——虽然3.5不是任何一次掷骰子的实际结果，但长期来看平均值会趋近于3.5。

**定理 29.6（期望的性质）**
1. $E(c) = c$（常数的期望为自身）
2. $E(aX + b) = aE(X) + b$（线性性）
3. $E(X + Y) = E(X) + E(Y)$（可加性，不要求独立）
4. 若 $X, Y$ 独立，则 $E(XY) = E(X)E(Y)$

**证明**：
1. $E(c) = c \cdot 1 = c$。

2. $E(aX + b) = \sum_k (ax_k + b)p_k = a\sum_k x_k p_k + b\sum_k p_k = aE(X) + b$。

3. 设 $(X, Y)$ 的联合分布为 $P(X = x_i, Y = y_j) = p_{ij}$，则：
$$E(X + Y) = \sum_{i,j}(x_i + y_j)p_{ij}$$

**绝对收敛性验证**：由 $E(X)$ 和 $E(Y)$ 存在（即 $\sum_i |x_i| P(X = x_i) < \infty$ 和 $\sum_j |y_j| P(Y = y_j) < \infty$），我们有：
$$\sum_{i,j} |x_i + y_j| p_{ij} \leq \sum_{i,j} (|x_i| + |y_j|) p_{ij} = \sum_i |x_i| \sum_j p_{ij} + \sum_j |y_j| \sum_i p_{ij} = \sum_i |x_i| P(X = x_i) + \sum_j |y_j| P(Y = y_j) < \infty$$

因此 $\sum_{i,j}(x_i + y_j)p_{ij}$ 绝对收敛，可以拆分求和：
$$E(X + Y) = \sum_i x_i \sum_j p_{ij} + \sum_j y_j \sum_i p_{ij} = \sum_i x_i P(X = x_i) + \sum_j y_j P(Y = y_j) = E(X) + E(Y)$$
$\blacksquare$

4. 若 $X, Y$ 独立，则 $E(XY) = E(X)E(Y)$。

**证明**：设 $P(X = x_i, Y = y_j) = p_{ij}$。独立性意味着 $p_{ij} = P(X = x_i)P(Y = y_j) = p_i^{(X)} p_j^{(Y)}$。

**绝对收敛性验证**：由 $E(X)$ 和 $E(Y)$ 存在，$\sum_i |x_i| p_i^{(X)} < \infty$ 和 $\sum_j |y_j| p_j^{(Y)} < \infty$，故：
$$\sum_{i,j} |x_i y_j| p_{ij} = \sum_{i,j} |x_i| |y_j| p_i^{(X)} p_j^{(Y)} = \left(\sum_i |x_i| p_i^{(X)}\right)\left(\sum_j |y_j| p_j^{(Y)}\right) < \infty$$

因此 $\sum_{i,j} x_i y_j p_{ij}$ 绝对收敛：
$$E(XY) = \sum_{i,j} x_i y_j p_i^{(X)} p_j^{(Y)} = \left(\sum_i x_i p_i^{(X)}\right)\left(\sum_j y_j p_j^{(Y)}\right) = E(X)E(Y)$$
$\blacksquare$

#### 29.4.3 方差

**定义 29.12（方差）** 随机变量 $X$ 的**方差**定义为：
$$D(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2$$

$\sqrt{D(X)}$ 称为 $X$ 的**标准差**。

**证明**：$D(X) = E[(X - E(X))^2] = E[X^2 - 2XE(X) + (E(X))^2] = E(X^2) - 2E(X) \cdot E(X) + [E(X)]^2 = E(X^2) - [E(X)]^2$。$\blacksquare$

> **方差的意义** 方差衡量随机变量取值的"离散程度"——方差越大，取值越分散。例如，两个投资方案的期望收益相同，但方差较大的方案风险更高。标准差与原变量同量纲，更便于直观理解。

**定理 29.7（方差的性质）**
1. $D(c) = 0$
2. $D(aX + b) = a^2 D(X)$
3. 若 $X, Y$ 独立，则 $D(X + Y) = D(X) + D(Y)$

**证明**：
1. $D(c) = E(c^2) - [E(c)]^2 = c^2 - c^2 = 0$。

2. $D(aX + b) = E[(aX + b - aE(X) - b)^2] = E[a^2(X - E(X))^2] = a^2 D(X)$。

3. $D(X + Y) = E[(X+Y)^2] - [E(X+Y)]^2 = E(X^2) + 2E(XY) + E(Y^2) - [E(X)]^2 - 2E(X)E(Y) - [E(Y)]^2$。

当 $X, Y$ 独立时，$E(XY) = E(X)E(Y)$（由定理 29.6 性质 4），故：
$$D(X + Y) = [E(X^2) - (E(X))^2] + [E(Y^2) - (E(Y))^2] = D(X) + D(Y)$$
$\blacksquare$

#### 29.4.4 常见离散分布

**定义 29.13（伯努利分布/0-1分布）** 随机变量 $X$ 只取 $0$ 和 $1$ 两个值：
$$P(X = 1) = p, \quad P(X = 0) = 1 - p \quad (0 < p < 1)$$

记作 $X \sim B(1, p)$。则 $E(X) = p$，$D(X) = p(1-p)$。

**定义 29.14（二项分布）** 在 $n$ 次独立伯努利试验中，成功次数 $X$ 服从**二项分布**：
$$P(X = k) = C_n^k p^k (1-p)^{n-k}, \quad k = 0, 1, \ldots, n$$

记作 $X \sim B(n, p)$。

> **直觉与动机** 二项分布是最重要的离散分布之一。它描述了"做 $n$ 次独立试验，每次成功概率为 $p$"的场景。例如，抛10次硬币正面朝上的次数、检验100件产品的合格数等。

**定理 29.8** 若 $X \sim B(n, p)$，则 $E(X) = np$，$D(X) = np(1-p)$。

**证明**：$X = X_1 + X_2 + \cdots + X_n$，其中 $X_i \sim B(1, p)$ 独立（$X_i$ 表示第 $i$ 次试验的结果）。

$$E(X) = \sum_{i=1}^{n} E(X_i) = np$$

$$D(X) = \sum_{i=1}^{n} D(X_i) = np(1-p)$$
$\blacksquare$

**定义 29.15（超几何分布）** 设 $N$ 个产品中有 $M$ 个次品，从中取 $n$ 个（不放回），次品数 $X$ 服从**超几何分布**：
$$P(X = k) = \frac{C_M^k C_{N-M}^{n-k}}{C_N^n}$$

**定义 29.16（泊松分布）** 随机变量 $X$ 服从参数为 $\lambda$（$\lambda > 0$）的**泊松分布**：
$$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}, \quad k = 0, 1, 2, \ldots$$

记作 $X \sim P(\lambda)$。则 $E(X) = \lambda$，$D(X) = \lambda$。

> **泊松分布的意义** 泊松分布描述稀有事件在固定时间段内发生的次数。例如，每分钟到达的顾客数、每页书中的错别字数、每平方公里内的交通事故数。当 $n$ 很大、$p$ 很小、$np = \lambda$ 适中时，二项分布 $B(n, p)$ 近似于泊松分布 $P(\lambda)$。
